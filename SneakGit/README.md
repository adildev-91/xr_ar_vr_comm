# Unity URP Project

This repository contains a Unity project configured with the Universal Render Pipeline (URP).

## Included
- Assets
- Packages
- ProjectSettings

## Excluded
The following folders are intentionally excluded via `.gitignore`:
- Library
- Temp
- Obj
- Build / Builds
- Logs
- UserSettings

These folders are automatically generated by Unity and will be recreated when the project is opened.

## Requirements
- Unity Hub
- Compatible Unity Editor version (as defined in ProjectSettings)
- Universal Render Pipeline package

## Notes
- First launch may take longer due to library regeneration.
